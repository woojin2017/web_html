CSS 기본

  1. class="" id="" - 지정해서 꾸며줄 때 사용
    style 작성 할 때
    class는 .class명{}
    id는 #id명{}으로 작성

  2. 예) a{ color:red;} 일 때
    a는 selector(태그,class,id)
    {}는 declaration(선언 또는 효과)를 줄 값을 지정
    color은 property 효과
    red는 vlaule(propety의 값) 어떤 효과를 줄건지

CSS 종류

  • 지정된 글의 색상 변경(빨간) - color:red;
  • 글 장식 (삭제) - text-decoration:none;
  • 글 크기 지정(45크기) - font-size: 45px;
  • 글 위치 지정(가운데) - text-align center;
  • 글 테두리 효과(위치 색 두께 선) - border-bottom: red 1px solid;
    (테두리 위치는 꼭 안해도 무방)
  • 화면에서 차지하는 공간 - display:
    grid; (격자)
    inline; (빈공간 삭제)
    block; (빈 공간 만들기)
    none; (안 보이게 하기)

그리드(Grid)

그리드를 사용하기 위해서 먼저 작성하기
display: grid;
grid-template-columns: 150px 1fr;

columns 열
rows 행

2개로 나눌 경우 크기 지정 두개 150px 1fr(비율)
1fr 1fr 1:1
1fr 2fr 1:2

grid 지정

  • 테두리와 콘텐츠 사이 20픽셀 padding: 20px;
  • 테두리와 테두리 사이 20픽셀 margin: 20px
  • padding - left: 20px; padding - right: 20px; 으로 한쪽만 가능 (margin도 같음)

미디어 쿼리

간단한 설명: c언어 java의 if문(조건문)
예시)
@media(max-width:800px){
#grid{
display:block;}

min은 최소 width > 800p이면 밑에 코드 실행
max는 최대 width < 800px이면 밑에 코드 실행
#grid는 위에 @media(조건)에 해당할 때 작동할 태그 class 또는 id 명
{} 안에는 동작 행동 또는 변경 할 것
